
# 🧪 Portfólio de Automação de Testes — Higor Damasceno Stella

Projeto de automação de testes front-end desenvolvido com **Cypress + JavaScript**, cobrindo cenários reais de autenticação, formulários, fluxos completos e validação de PDF.

---

## 🚀 Tecnologias Utilizadas

- [Cypress](https://www.cypress.io/) — Framework de testes end-to-end
- JavaScript — Linguagem principal dos testes
- Node.js — Ambiente de execução
- Git & GitHub — Controle de versão

---

## 📁 Estrutura do Projeto

```
├── web/          # Testes de interface (front-end)
├── api/          # Testes de API
└── README.md
```

---

## ✅ O que está coberto

### 🔐 Autenticação
- Login com credenciais válidas
- Login com credenciais inválidas
- Validação de mensagens de erro

### 📝 Formulários
- Preenchimento e envio de formulários
- Validação de campos obrigatórios
- Verificação de feedback ao usuário

### 🔄 Fluxos Completos
- Fluxo de cadastro de usuário
- Navegação entre páginas e validação de estados

### 📄 Validação de PDF
- Verificação de geração e download de arquivos PDF

---

## ▶️ Como Executar

### Pré-requisitos
- Node.js 18+
- npm ou yarn

### Instalação

```bash
# Clone o repositório
git clone https://github.com/Higormu2/Portfolio.git

# Acesse a pasta
cd Portfolio

# Instale as dependências
npm install
```

### Rodando os testes

```bash
# Abre a interface gráfica do Cypress
npx cypress open

# Executa os testes em modo headless
npx cypress run
```
